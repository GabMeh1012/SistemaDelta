# Casos de Prueba — SistemaDelta (corregidos)

Verificados contra el código fuente y, en 3 casos, contra la base de datos en vivo (transacciones de
prueba revertidas). Se corrigieron 3 discrepancias reales confirmadas con el equipo (CP-010, CP-019,
CP-021), 2 detalles menores de redacción (CP-003, CP-027), y 11 casos donde los "Pasos"/"Módulo" usaban
nombres de pestaña o botón que no coinciden con el sistema real (CP-009, CP-010, CP-011, CP-012, CP-013,
CP-016, CP-017, CP-018, CP-019, CP-022, CP-026) — verificados directamente contra las tres JSP
(`portal_estudiante.jsp`, `portal_profesor.jsp`, `portal_administrador.jsp`).

conserva con estado 'retirado') y se agregaron 6 casos nuevos por funcionalidad de nuevo desarrollo
(CP-033 a CP-038, dejando libres CP-031 y CP-032 para los casos de navegación/sesión de `index.jsp`

## Notas de corrección — revisión 10/07/2026

**CP-021:** el hallazgo original de este caso ("inserta un registro nuevo en vez de actualizar") ya no es válido — `AsistenciaDAO.guardar` usa `INSERT ... ON DUPLICATE KEY UPDATE` sobre la clave (inscripción, fecha), que sí actualiza correctamente. El bug real detectado por el equipo esta semana era otro: `cycleAttDay` y `saveAttendance` en `portal_profesor.jsp` calculaban la fecha de "hoy" con `new Date().toISOString().slice(0,10)` — al convertir a UTC antes de recortar la fecha, en Panamá (UTC-5) cualquier hora local después de las 7pm caía en el día UTC siguiente, así que el cambio se guardaba bajo la clave de fecha equivocada y nunca aparecía en la fila de hoy de "Asistencia de la Semana" (que sí usa la fecha local correcta). Corregido centralizando el cálculo en una función `localDateKey()` que primero pone la hora en 0 en local antes de convertir.

**CP-025:** el paso original ("escribir el nombre sin tildes y verificar que el sistema lo encuentre") ya no aplica — desde el cambio de "Mensajería restringida por carrera", el campo de destinatario dejó de ser un input de texto libre con búsqueda y pasó a ser un `<select>` con las opciones válidas ya cargadas (ver §7 de este documento y el commit `acf5b15`). Se reescribió el caso para probar lo que existe hoy.
